7. CORE ROUTER — src/core/router/
======================================================================================================

  core/router.py — class APIRouter(KeyResolverMixin)
  ├── _get_custom_ep() — Lấy custom endpoint config
  ├── freeze_key() — Adaptive cooldown + penalty jitter + per-model freeze
  ├── freeze_all_keys() — Global freeze (circuit breaker)
  ├── record_success() — Reset failures, update last_success, fire DB write
  ├── record_429() — Global circuit breaker (15 429s → block 10-20s)
  ├── release_key() — active_requests-- + fire-and-forget DB write
  └── reserve_key() — Key selection → atomic_reserve_key → record_key_model_usage

  core/key_resolver.py — class KeyResolverMixin
  ├── _key_is_circuit_open() — Check circuit breaker
  ├── _adaptive_cooldown() — 429: exponential backoff 3^(n-1) capped 600s
  ├── _load_key_tiers() — Load tier từ DB
  ├── _get_allowed_tiers() — Tiers allowed cho account
  └── reserve_key() — Iterate keys → filter frozen/busy → priority rank → pick → reserve

  pool.py — class ModelPool
  ├── Pool state machine (transient vs hard failure)
  ├── Swap threshold tracking
  └── Auto-swap logic (flash-35 → flash-30 → flash-25)

  DEPENDENCIES:
    router.py ──> key_resolver.py
    key_resolver.py ──> backend/key_status.py (DB operations)
    key_resolver.py ──> core/limits/gemini_rate_limiter.py
    key_resolver.py ──> core/config_n_logg (config, logger)

    ỨNG DỤNG ADAPTIVE COOLDOWN:
    ──────────────────────────────────────────────────────────────────────────────
    - 429 (rate limit):     exponential backoff 3^(n-1), base 30s, capped 600s
    - RPD vượt quá giới hạn:  freeze đến midnight Pacific + 300s
    - timeout:              60s * 2^(min(cf-1,3))
    - Circuit breaker:       global block 10-20s (sau 15 lần 429 liên tiếp)
    - invalid_key:           DISABLED vĩnh viễn (atomic_disable_key)
    - permission_denied:     DISABLED vĩnh viễn (atomic_disable_key)
    - Jitter ngẫu nhiên:     0-15% + gaussian margins
    - Giảm ưu tiên key:     penalty score khác nhau theo lỗi (-20 đến -86)
    - Per-model freeze:      freeze riêng model + short global freeze (max 12s)
